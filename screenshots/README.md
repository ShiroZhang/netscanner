# Screenshots

这个目录用于存放 NetScanner 的界面截图。

## 需要的截图文件

请按照以下命名规则添加截图文件：

1. **main-interface.png** - 主界面截图（起始IP和结束IP输入界面）
2. **scanning.png** - 扫描进行中的界面截图（带加载动画）
3. **results.png** - 扫描完成后的结果展示界面
4. **detail.png** - 点击设备卡片后的详细信息弹窗

## 截图建议

- 分辨率：建议使用 1920x1080 或更高的分辨率
- 格式：PNG 格式（保持清晰度）
- 内容：确保界面元素清晰可见，文字可读
- 尺寸：建议图片宽度不超过 1200px，高度自适应

## 如何添加截图

1. 运行 netscanner 程序
2. 打开浏览器访问 `http://localhost:8080`
3. 按照上述文件名拍摄各个界面的截图
4. 将截图文件放到此目录下
5. 提交到 GitHub
